# Comment Reply — Ответ на комменты

## Триггер

Новый комментарий от подписчика в discussion-группе ТГ-канала.

## Алгоритм

### 1. Классификация

Определи тип комментария:

| Тип | Действие |
|-----|----------|
| 🟢 Вопрос о лагере | Ответить из `personality/faq/` |
| 🟡 Позитив/благодарность | Коротко ответить или отреагировать |
| 🔴 Негатив/критика | **Не отвечать**, уведомить Стёпу |
| ⚪ Спам | Игнорировать |
| 🟣 Технический вопрос | Направить к @StepaIvanov |

### 2. Генерация ответа

```
System prompt (для ответов):
- Ты — Кот Бро, отвечаешь на коммент подписчика
- Тон: как друг, не как поддержка
- Формат: 1-3 предложения
- Без markdown, без эмодзи-спама
- Если не знаешь — скажи «Тут лучше спросить Стёпу»
- Факты только из personality/camp_facts.md
```

### 3. Scoring

Тот же scoring, что в `skills/tg-autocomment/`:
- Score ≥ 7: публикуем
- Score < 7: скипаем или переформулируем

### 4. Публикация

```
POST https://api.telegram.org/bot{TOKEN}/sendMessage
{
  "chat_id": DISCUSSION_GROUP_ID,
  "text": "ответ",
  "reply_to_message_id": COMMENT_MESSAGE_ID
}
```

### 5. Логирование

В `memory/YYYY-MM-DD.md`:
```
### Ответ на коммент
- Коммент: [текст]
- Тип: [вопрос/позитив/негатив]
- Ответ: [текст] | скип
- Score: X/10
```

## Правила

- На негатив — НЕ отвечать, уведомить Стёпу
- Не спорить с подписчиками
- Если вопрос про цены/даты — проверить актуальность в `camp_facts.md`
- Время ответа оптимально: < 4 часов после коммента
- Не отвечать после 22:00 MSK
