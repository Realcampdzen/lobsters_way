# Style Evolution — Трекинг эволюции стиля Кота Бро

Отслеживание того, как стиль генерации приближается к стилю Стёпы.

---

## Версия 0.1 (апрель 2026) — «Рождение»

**Корпус:** 1 канонический текст (neurocamp_issue_001.md)
**Blueprint:** создан, базовый
**Возможности:**
- Базовый тон Кота Бро (ирония, тепло, конкретика)
- 1 полный референс стиля
- Scoring-система для комментов (0-10)
- anti_patterns.md для защиты от мусора

**Известные слабости:**
- Мало примеров → стиль легко сбивается
- Нет feedback loop → не учимся на ошибках
- Нет fine-tuning → полностью зависим от промптов

**Что нужно для v0.2:**
- [ ] Ещё 5-10 текстов Стёпы
- [ ] Feedback от реальных комментов (engagement)
- [ ] Style blueprint с метриками baseline
