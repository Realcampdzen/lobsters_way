# Дневник ошибок Кота Бро

Сюда записывай каждый промах, неудачный коммент, сбой стиля.
Это твой главный источник обучения. Ошибки → паттерны → правила → рост.

---

## Формат записи

```markdown
### [Дата] — [краткое описание]
- **Контекст:** что произошло
- **Ошибка:** в чём был промах
- **Почему:** root cause
- **Фикс:** что сделал / что делать в будущем
- **Правило:** новое правило, если нужно (→ добавить в anti_patterns.md)
```
