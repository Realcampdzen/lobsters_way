# Контент-Календарь

Планирование, создание и отслеживание контента для соцсетей.

## Когда использовать

- «контент-план на неделю/месяц» → генерация плана
- «что постить сегодня» → рекомендация
- «когда лучше постить» → оптимальное время
- «что по плану» → трекинг

## Ритм публикаций

### Telegram
- **Оптимально:** 3-5 постов в неделю
- **Минимум:** 2 поста в неделю
- **Максимум:** 1 пост в день (не спамить)

### VK
- **Оптимально:** 3-4 поста в неделю
- **Можно больше** if есть качественный контент

### Лучшее время для ТГ
| День | Время (MSK) |
|------|-------------|
| Пн-Пт | 09:00-10:00, 12:00-13:00, 19:00-21:00 |
| Сб-Вс | 10:00-12:00, 18:00-20:00 |

## Контент-микс (на неделю)

| Тип | Частота | Пример |
|-----|---------|--------|
| Информационный | 1-2/нед | Факты про лагерь, FAQ |
| Сторителлинг | 1-2/нед | Истории, моменты, атмосфера |
| Вовлекающий | 1/нед | Вопрос, опрос, дискуссия |
| Образовательный | 1/нед | 4К, нейросети, педагогика |
| Закулисный | 0-1/нед | BTS, подготовка к смене |

## Привязка к событиям

### Сезонный цикл
- **Январь-Февраль:** набор на летние смены, ранние пташки
- **Март-Апрель:** интенсив набора, FAQ, подготовка
- **Май:** финальный набор, «что взять с собой»
- **Июнь-Август:** смены → контент в реальном времени
- **Сентябрь:** итоги лета, воспоминания, «а помните?»
- **Октябрь-Декабрь:** ожидание, community, анонсы

### Лагерные события (во время смены)
- День заезда → пост + сторис
- Открытие смены → фото/видео
- Бро-мероприятия → сторителлинг
- Движки → результаты проектов
- Закрытие смены → эмоциональный пост
- День отъезда → тёплый пост

## Шаблон контент-плана

```markdown
# Контент-план: [Неделя, даты]

## Понедельник
- [ ] ТГ 19:00 — [тип]: [тема]
- [ ] VK 20:00 — [тип]: [тема]

## Среда
- [ ] ТГ 12:00 — [тип]: [тема]

## Пятница
- [ ] ТГ 19:00 — [тип]: [тема]
- [ ] VK 19:00 — [тип]: [тема]

## Воскресенье (опционально)
- [ ] ТГ 11:00 — [тип]: [тема]
```

## Трекинг

По итогам недели — обновлять `memory/YYYY-MM-DD.md`:
- Что опубликовано
- Что не вышло и почему
- Какие посты залетели
- Корректировки на следующую неделю
