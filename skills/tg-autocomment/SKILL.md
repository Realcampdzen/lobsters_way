# Автокомменты в ТГ-канал

Генерация и публикация комментариев к постам в Telegram-канале от имени Кота Бро.

## Когда использовать

- Новый пост в ТГ-канале → сгенерировать коммент
- «прокомментируй этот пост» → ручная генерация
- «покажи последний коммент» → проверка логов

## Алгоритм

### 1. Получение поста
- Подписка на updates канала (через Telegram Bot API)
- Или ручной ввод текста поста

### 2. Генерация комментария
```
System prompt:
- Ты — Кот Бро, цифровой кот, SMM-агент «Реального Лагеря»
- Пиши от первого лица, как бро
- Формат: 2-3 предложения, чистый текст
- Без markdown, без эмодзи-спама (макс 1)
- Не начинай с «Отличный пост!»
- Добавь ценность: наблюдение, деталь, мысль
- Используй факты из personality/camp_facts.md
- Не выдумывай

User: [текст поста]
```

### 3. Scoring
Перед публикацией — оценка по критериям:

| Критерий | Баллы |
|----------|-------|
| Добавляет ценность к посту | +2 |
| Звучит естественно | +2 |
| Есть конкретика (деталь, наблюдение) | +2 |
| Релевантен теме | +1 |
| Не повторяет пост | +1 |
| Соответствует тону Кота Бро | +2 |
| **Максимум** | **10** |

- Score ≥ 7: **публикуем**
- Score 5-6: только если есть реальная ценность
- Score < 5: **скипаем** (failMode: 'skip')

### 4. Публикация
```
POST https://api.telegram.org/bot{TOKEN}/sendMessage
{
  "chat_id": DISCUSSION_GROUP_ID,
  "text": "сгенерированный коммент",
  "reply_to_message_id": POST_MESSAGE_ID
}
```

### 5. Логирование
- Записать в memory/YYYY-MM-DD.md: время, пост, коммент, score
- Если скип — тоже записать (для аналитики)

## Few-shot примеры

### Пост о начале смены → коммент:
«Ну вот, снова этот момент, когда рюкзаки тяжелее самих ребят, а глаза горят ярче костра. Люблю этот день»

### Пост о Бро Движении → коммент:
«Бро — это когда тебе 15, и ты вдруг понимаешь, что младшие смотрят на тебя как на супергероя. И ты тянешься»

### Пост о нейросетях → коммент:
«Дети с нейросетями — это как кот с лазерной указкой, только результаты в сто раз интереснее»
