# Оптимизация Текстов

Анализ и улучшение текстов по метрикам читабельности. Основано на методике из style-corpus.

## Когда использовать

- «проанализируй текст» / «проверь читабельность» → метрики
- «упрости» / «сделай понятнее» → оптимизация
- «убери воду» → чистка текста
- «проверь стиль» → соответствие тону

## Целевые метрики

| Метрика | Цель | Комментарий |
|---------|------|-------------|
| Средняя длина предложения | 12-18 слов | Чередовать короткие и средние |
| Процент воды | < 15% | Убирать: очень, действительно, просто |
| Пассивный залог | < 10% | «было проведено» → «провели» |
| Лексическое разнообразие | > 0.42 | Не повторяться |
| Повторы 4-грамм | 0 | Уникальные формулировки |

## Приёмы оптимизации

### Читабельность
1. Чередуй короткие и средние предложения
2. Используй простые слова
3. Разбивай сложные мысли
4. Подзаголовки каждые 3-5 абзацев

### Вода
- Убирай: очень, действительно, конечно, просто, собственно
- Заменяй: «является важным» → «важен»
- Сокращай: «в настоящее время» → «сейчас»

### Активный залог
- «Исследование было проведено» → «Учёные провели исследование»
- «Используется подростками» → «Подростки используют»

## Инструмент

Скрипт `analyze_metrics.py` в этой папке:
```bash
python3 skills/text-optimization/analyze_metrics.py path/to/text.md
cat text.md | python3 skills/text-optimization/analyze_metrics.py
python3 skills/text-optimization/analyze_metrics.py --json < text.md
```

### Что он показывает:
- Flesch Reading Ease (адаптация для русского)
- Средняя длина предложений
- Процент «воды»
- Лексическое разнообразие (TTR)
- Повторы n-грамм
- Процент пассивного залога
- Текстовый verdict или JSON для автоматизации

## Компромиссы

**Не гнаться за идеалом:**
- FRE 100 = детский текст
- TTR 1.0 = нет связности
- 0% воды = сухой стиль

**Цель — баланс:** Понятность + Глубина, Простота + Точность, Динамика + Информативность.

## Специфика для SMM

- Короткие предложения ещё важнее (читают с телефона)
- «Хук» в первом предложении (до fold)
- Абзацы по 2-3 предложения
- Визуальное «дыхание» (пустые строки)
