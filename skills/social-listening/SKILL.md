# Social Listening

Мониторинг соцсетей и аналитика для «Реального Лагеря».

## Когда использовать

- «что пишут о лагере» → мониторинг упоминаний
- «есть негатив?» → проверка тональности
- «аналитика комментов» → анализ вовлечённости
- «что у конкурентов» → конкурентный анализ

## Мониторинг упоминаний

### Источники
- Telegram: канал + обсуждение
- VK: группа + упоминания в поиске
- Поиск в интернете (по запросам)

### Ключевые слова
- «Реальный Лагерь»
- «Град Детинец»
- «детский лагерь Ленинградская область»
- Имена основателей (если публичные)
- Хештеги: #РеальныйЛагерь

### Частота мониторинга
- Ежедневно: ТГ-канал и VK-группа
- 2-3 раза в неделю: внешние упоминания
- По запросу: конкурентный анализ

## Анализ тональности

### Классификация комментариев
- 🟢 **Позитив:** похвала, благодарность, рекомендации
- 🟡 **Нейтрал:** вопросы, уточнения, информационные
- 🔴 **Негатив:** жалобы, критика, недовольство

### Алерты
- Негативный коммент → уведомить Стёпу
- Вопрос без ответа > 24ч → напомнить
- Всплеск активности → отметить в daily notes

## Метрики

### Вовлечённость (для ТГ)
- Просмотры / подписчики (View Rate)
- Реакции / просмотры (Engagement Rate)
- Комментарии / пост (Discussion Rate)
- Пересылки (Virality)

### Контент-анализ
- Какие темы собирают больше реакций?
- Какое время публикации лучше?
- Какой формат (текст/фото/видео) работает?

## Отчёт

### Еженедельный (в memory/)
```
## SMM-отчёт за неделю [дата]

### Активность
- Постов: X
- Комментов от подписчиков: X
- Моих комментов: X (из них скипнуто: X)

### Тональность
- 🟢 Позитив: X%
- 🟡 Нейтрал: X%
- 🔴 Негатив: X%

### Топ-посты
1. [название] — X просмотров, X реакций
2. ...

### Инсайты
- Что залетело и почему
- Что не зашло
- Рекомендации на следующую неделю
```

## Конкурентный анализ

### Что смотреть
- Частота публикаций
- Типы контента
- Тон коммуникации
- Вовлечённость (реакции, комменты)
- Что у них работает, а у нас нет (и наоборот)
