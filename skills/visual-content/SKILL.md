# Визуальный Контент

Генерация и рекомендации по визуальному контенту для SMM.

## Когда использовать

- «картинка для поста» → промпт для генерации
- «обложка» / «сторис» → формат + рекомендации
- «стиль визуала» → брендбук-лайт
- «подпись к фото» → caption

## Форматы

| Формат | Размер | Где |
|--------|--------|-----|
| Пост (квадрат) | 1080×1080 | VK, ТГ |
| Пост (горизонт) | 1200×630 | VK, ТГ |
| Сторис/Reels | 1080×1920 | VK Stories |
| Обложка ТГ | 1280×720 | Telegram |
| Аватар | 512×512 | Везде |

## Цветовая палитра бренда

### Основные цвета
- **Основной:** Тёплый оранжевый (#FF7043) — энергия, дружба
- **Акцент:** Глубокий синий (#1565C0) — доверие, стабильность
- **Фон:** Тёплый белый (#FFF8F0) — чистота, открытость

### Дополнительные
- **Зелёный:** #43A047 — природа, лето
- **Фиолетовый:** #7E57C2 — креатив, нейросети
- **Жёлтый:** #FDD835 — радость, детство

### Не использовать
- Неоновые режущие цвета
- Чёрный фон (не соответствует атмосфере)
- Серый как основной (скука)

## Стиль изображений

### Фотографии (реальные)
- Яркие, солнечные, с людьми
- Эмоции > позирование
- Природа, лагерь, активности
- Без постановки — живые моменты

### Иллюстрации (сгенерированные)
```
Стиль: тёплая акварельная иллюстрация
Цвета: оранжевый, синий, зелёный
Атмосфера: летняя, дружеская, позитивная
Персонажи: подростки 12-16 лет
Сеттинг: природа, лагерь, костёр, мастерские
НЕ: фотореализм, аниме, 3D-рендер
```

## Подписи к фото (captions)

### Структура
1. **Хук** (первая строка — самая важная)
2. **Контекст** (что на фото, кратко)
3. **Эмоция/мысль** (от лица Кота Бро)
4. **CTA** (только для постов, не для комментов)

### Примеры
```
Хук: «Этот момент на сцене...»
Контекст: «Когда репетируешь три дня и наконец выходишь.»
Эмоция: «Кот Бро уважает смелость.»
```

## Шаблоны текстов на картинках

### Для постов
- Шрифт: чистый sans-serif
- Цвет текста: белый на тёмном, оранжевый на светлом
- Максимум 6-8 слов на картинке
- Контраст > красота

### Не использовать
- Курсивные/рукописные шрифты (нечитаемо с телефона)
- Мелкий текст
- Текст поверх лиц
