# Создание Контента

Скилл для генерации SMM-контента для «Реального Лагеря».

## Когда использовать

- «придумай пост» / «напиши пост» → генерация текста
- «идеи для контента» → brainstorm
- «заголовки» / «варианты заголовков» → A/B заголовки
- «адаптируй под ТГ/VK/Reels» → кросс-платформа

> Новые посты по умолчанию создаются как draft. Автопубликация контент-постов не входит в этот репозиторий.

## Форматы контента

### Telegram-пост
- **Длина:** 500-1500 символов (оптимально — 800)
- **Структура:** Хук → Основная мысль → Завершение
- **Форматирование:** HTML-разметка (bold, italic, ссылки)
- **Эмодзи:** 2-4 на пост, структурно (в начале абзацев)

### VK-пост
- **Длина:** 500-2000 символов
- **Структура:** Заголовок → Текст → Вопрос/CTA
- **Хештеги:** 3-5 на пост в конце
- **Вложения:** фото/видео рекомендация

### Reels/Stories (текст для подписи)
- **Длина:** 100-300 символов
- **Стиль:** Короткий, энергичный, с хуком
- **Хештеги:** 5-10 для reach

## Типы постов

### 1. Информационный
Факты о лагере, программа, даты, ответы на вопросы.
```
Бери факты из: personality/camp_facts.md
Тон: информативный, но живой
```

### 2. Сторителлинг
Истории из жизни лагеря, моменты, сцены.
```
Тон: тёплый, наблюдательный
Фокус: атмосфера, эмоции, детали
НЕ выдумывай конкретные случаи — обобщай
```

### 3. Вовлекающий
Вопросы, опросы, «а помните?», дискуссии.
```
Тон: дружеский, открытый
Формат: вопрос в конце
```

### 4. Образовательный
4К-навыки, нейросети, педагогика — полезный контент.
```
Тон: доступный, без академизма
Формат: проблема → решение → пример
```

### 5. Закулисный
Behind the scenes, как устроен лагерь, что делает команда.
```
Тон: искренний, неформальный
```

## A/B заголовки

При генерации заголовков — предлагай 3 варианта:
1. **Прямой** — чётко по теме
2. **Интригующий** — с хуком
3. **Эмоциональный** — через чувства
