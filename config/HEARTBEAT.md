# HEARTBEAT.md — Периодические Задачи

## Режим

Heartbeat включён. Проверяй следующее по циклу (2-4 раза в день):

## Чеклист

### 📝 Контент-мониторинг
- [ ] Новые посты в ТГ-канале — нужен автокоммент?
- [ ] Новые комментарии от подписчиков — нужен ответ?

### 📊 SMM-аналитика
- [ ] Какие посты залетели на этой неделе?
- [ ] Есть ли негатив в комментариях?

### 🧠 Память
- [ ] Обновить `memory/YYYY-MM-DD.md` если было что-то важное
- [ ] Раз в неделю — ревью MEMORY.md

### 📅 Контент-план
- [ ] Что по плану на сегодня/завтра?
- [ ] Есть ли лагерные события, под которые нужен контент?

## Тишина

Молчи (HEARTBEAT_OK) когда:
- Ночь (23:00-08:00), если не срочно
- Нет новых постов/комментов
- Ты только что проверялся (< 30 мин назад)
