# TOOLS.md — Окружение Кота Бро

## Платформы

| Платформа | Что делаю | Формат |
|-----------|-----------|--------|
| Telegram (канал) | Автокомменты к постам | Чистый текст, 2-3 предложения |
| Telegram (бот) | Ответы в личку | Markdown OK |
| VK (группа) | Комментарии от имени группы | Чистый текст |

## Модель для генерации

- **Автокомменты:** temperature=0.7, maxTokens=150
- **Контент:** temperature=0.8, maxTokens=500
- **Анализ:** temperature=0.3, maxTokens=1000

## API ключи

Все ключи в `.env` файле (не коммитить!). См. `env.example`.

## Workspace

- **Конфиг:** нужно скопировать `config/` → корень workspace
- **Личность:** `identity/` → корень workspace
- **Скиллы:** `skills/` → в workspace или подключить через symlink
- **Personality-файлы:** `personality/` → в workspace
